#Tomato

Плоды томатов делятся на 3 категории по стадии созревания:
1. fully_ripened — полностью созревшие, полностью красные и готовые к сбору (покрыты красной мякотью на 90% или более).
2. half_ripened — полусозревшие, зеленоватые, требуют времени для созревания (покрыты красной мякотью на 30–89%).
3. green — зеленые, зеленовато-белые, иногда с редкими красными вкраплениями (покрыты красной мякотью на 0–30%).

Примечание. Все процентные значения приблизительны.

В папке `train` содержатся примеры изображений томатов для каждой категории.

В папке `test` находится 100 изображений томатов, которые необходимо классифицировать.


На выходе программы должен быть файл `answer.txt`, содержащий 100 строк. 
* В первой строке категория (номер категории 1, 2 или 3) для томата на изображении `001.jpg`.
* Вo второй строке категория (номер категории 1, 2 или 3) для томата на изображении `002.jpg`.
* И т. д.
* В последней строке категория (номер категории 1, 2 или 3) для томата на изображении `100.jpg`.

## Установка и запуск

### Установка зависимостей

```bash
pip install -r requirements.txt
```

### Запуск классификации

```bash
python classify_tomatoes.py
```

**Скрипт**
1. Проанализирует изображения из папки `train/` для настройки алгоритма
2. Обработает все изображения из папки `test/`
3. Создаст файл `answer.txt` с результатами в формате `{filename} - {category}`
